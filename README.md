# awesome-idml
A curated awesome list for IDML (Adobe InDesign Markup Language): open-source libraries (Python, PHP, JS/TS, Rust, Java, XSLT), converters, apps that open &amp; edit IDML (InDesign, Affinity, QuarkXPress, Scribus, VivaDesigner), database publishing, localization tools, and format docs. Everything for working with .idml files.
